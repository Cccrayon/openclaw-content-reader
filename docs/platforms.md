# 各平台详细说明

## 设计理念

**让用户先用起来，碰到墙了再给梯子。**

- 不提前要求登录配置
- 用到时再提示可选项
- 默认使用无需登录的方案

---

## YouTube

### 特点
- ✅ **最简单**：无需登录，字幕直接可下载
- ✅ **速度快**：字幕下载通常只需几秒
- ✅ **覆盖广**：几乎所有视频都有英文自动字幕

### 工作流程

```
发送链接 → 尝试字幕下载 → 【成功】输出 【失败】Whisper 兜底
```

### 字幕获取

```bash
# 查看可用字幕
yt-dlp --list-subs "URL"

# 下载字幕（优先级：英文 → 中文 → 其他）
yt-dlp --write-sub --write-auto-sub --sub-lang en,zh-Hans \
  --skip-download -o "/tmp/yt/%(id)s" "URL"
```

## 抖音

### 特点
- ✅ **完全独立**：无需登录，无需 API Key
- ✅ **全自动化**：脚本一键完成

### 工作流程

```
发送链接 → 自动解析URL → 下载视频 → ffmpeg提取音频 → Whisper转写
```

### 脚本参数

```bash
python3 scripts/douyin_whisper.py [OPTIONS]

Options:
  -m, --model      Whisper 模型 (tiny/base/small/medium) [默认: base]
  -l, --language   语言代码 (zh/en等) [默认: zh]
  -o, --output     输出文件路径
  -k, --keep       保留中间文件（视频+音频）
```

## B站（哔哩哔哩）

### 特点
- ✅ **默认 AI 字幕**：无需登录，已足够日常使用
- ⚠️ **可选 Cookie**：需要更高质量时再配置

### 工作流程

```
发送链接 → 尝试 AI 字幕 → 【成功】输出 【失败】Whisper 兜底
                                              ↓
                                         可选：配置 Cookie 获取官方字幕
```

### 字幕方案对比

| 方案 | 需要 Cookie | 质量 | 速度 |
|------|-------------|------|------|
| AI 字幕 (ai-zh) | ❌ | 中等 | 快 |
| 官方字幕 | ✅ | 高 | 快 |
| Whisper 转写 | ❌ | 中等 | 慢 |

### Cookie 配置（可选）

**什么时候需要？**
- AI 字幕理解困难时
- 需要特定语言（如粤语、日语）官方字幕时

**如何配置：**
```bash
# 1. 浏览器登录 bilibili.com
# 2. 安装 EditThisCookie 扩展
# 3. 导出 Cookie 为 Netscape 格式
# 4. 保存到 /tmp/bili_cookies.txt
```

**如果不想配置？**
- 完全没问题，AI 字幕已足够日常使用

## 小红书

### 特点
- ✅ **图文完整**：正文+图片+标签
- ✅ **无需登录**
- ⚠️ **可选 OCR**：需要图片文字识别时配置 MiniMax API

### 工作流程

```
发送链接 → browser打开 → JS滚动加载 → 提取内容
                                      ↓
                              可选：MiniMax OCR 图片文字识别
                                      ↓
                                   输出
```

### MiniMax OCR 配置（可选）

```bash
# 安装 mcporter
npm install -g mcporter

# 配置环境变量
# MINIMAX_API_KEY=your_key
```

## 推荐使用场景

| 场景 | 推荐 | 原因 |
|------|------|------|
| 快速获取 | YouTube | 字幕直接下，无需等待 |
| 中文视频 | 抖音/B站 | 语音识别更准 |
| 图文内容 | 小红书 | 支持图文+图片 |
| 最佳效果 | YouTube/抖音 | Whisper 转写质量高 |
